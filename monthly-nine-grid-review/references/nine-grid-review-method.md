# Monthly Nine-Grid Review Method

当用户要做每月工作、生活等九宫格复盘，或需要把滴答清单、RescueTime、ima 笔记映射到生活结构时，使用此参考。此方法借鉴邹小强《只管去做》中从愿景到年度九宫格、再到月/周/日行动的落地思路。

## 核心原则

九宫格不是九个愿望清单，而是九个生活系统仪表盘。每个格子都要回答：

- 本月发生了什么事实？
- 投入了多少时间、精力或金钱？
- 产出了什么可见结果？
- 偏差来自能力、系统、资源、外部依赖，还是目标本身不重要？
- 下月要保持、停止、简化、推进什么？

## 只管去做式月度校准

月复盘的重点不是重新发愿，而是校准年度方向有没有落到行动：

1. 回看年度愿景或年度九宫格：本月是否仍在同一方向上。
2. 检查本月九宫格：哪些格子有证据，哪些只是想象。
3. 拆成三类承诺：结果目标、项目推进、习惯维持。
4. 把下月重点倒推到第一周，再落到每日最小动作。
5. 用 4D 清理任务池：立刻做、计划做、委托、删除。
6. 用真、小、实、少过滤计划：是不是我真想要；能不能切小；是否有具体动作；数量是否过载。

## 默认九宫格

优先使用用户已有分类。没有现成分类时，使用：

| 格子 | 典型问题 | 证据来源 |
|---|---|---|
| 工作/事业 | 本月真正创造了什么职业价值？ | OKR、项目任务、会议笔记、RescueTime |
| 个人项目/OKR | 哪些长期目标有实质推进？ | KR 进度、TickTick、周复盘 |
| 学习成长 | 哪项能力变强了？有没有输出？ | ima 笔记、课程、读书笔记、博客、Anki |
| 健康精力 | 身体是否支持目标？是否透支？ | 习惯、运动、睡眠、工作时长 |
| 家庭/亲密关系 | 重要关系有没有被真实投入？ | 日历、任务、笔记 |
| 社交/合作 | 本月的连接是否高质量？ | 会议、社群、朋友、合作机会 |
| 财务/资产 | 收支、投资、预算、风险是否清楚？ | 财务记录、任务、笔记 |
| 环境/物品/系统 | 生活和数字环境是否更轻？ | 文件、物品、订阅、自动化、Someday |
| 心智/情绪/乐趣 | 压力、意义感、快乐来源是否健康？ | 日记、时间日志、休闲记录 |

## 每格评分

给每个格子 1-5 分：

- 5：明显超预期，有可复用机制。
- 4：达成主要预期，有小问题。
- 3：基本维持，但没有明显进展。
- 2：低于预期，存在系统性阻塞。
- 1：失控或长期忽视，需要下月明确干预。

评分后必须写出一句证据，不接受只有感受的分数。

## 每格输出

```markdown
### 格子名
- 评分：
- 本月事实：
- 证据：
- 年度方向关联：
- 超预期 / 正常 / 不符预期：
- 根因：
- 结果 / 项目 / 习惯：
- 保持：
- 停止 / 简化：
- 下月推进项：
- 第一周行动：
- 每日最小动作：
```

## 汇总方式

完成九格后，再做三类横向总结：

- 杠杆格：本月最能带动其他格子的 1-2 个领域。
- 漏水格：持续消耗精力但没有产出的 1-2 个领域。
- 修复格：下月必须最低限度修复的 1 个领域。

下月计划优先围绕杠杆格和修复格，而不是平均用力。

## 下月计划门槛

- 每个格子最多 1 个主推进项；维护类格子可以没有推进项，但要写清维护动作。
- 每个推进项必须能进入滴答清单或日历。
- 每个推进项必须能在第一周启动，不接受只写「继续提升」「多关注」。
- 如果计划超过可用时间，优先删减低杠杆格，而不是平均压缩每件事。
