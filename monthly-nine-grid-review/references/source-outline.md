# Source Outline

月度九宫格复盘 Skill 保留用户原始方案：

1. 复盘
   - OKR 进度 from TickTick/Dida summary.
   - 九宫格填写.

2. 极简
   - 哪些事情不需要做.
   - 哪些物品不再需要.

3. 规划
   - 已有日程更新.
   - 现有规划细化.
   - 每个 KR 当月都有推进项.

扩展后的 Skill 将其变成月度个人运营闭环：

- 用滴答清单处理承诺、完成情况、逾期项、习惯、OKR 和下月任务草案。
- 用 RescueTime 分析注意力分配和低价值时间消耗。
- 用 ima copilot 和个人笔记提取决策、未闭环事项、洞见和九宫格证据。
- 围绕工作、生活、健康、关系、学习成长、财务、环境、心智等九宫格形成月度判断。
- 借鉴邹小强《只管去做》的愿景、年度九宫格、SMART、项目计划、习惯和 4D 任务处理思路。
- 最终产出 not-to-do 决策、下月 KR 推进项、第一周行动和风险预判。
