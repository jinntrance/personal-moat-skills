# Source Outline

This reference preserves the user's original weekly review structure in compressed form.

## Get Clear

- Finish unresolved cleanup, archive, and index.
- Put internally remembered items and externally assigned TODOs into Inbox; clear Inbox.
- Add preparation tasks from future calendar events.

## Get Current

- Time logs: RescueTime, iPhone Screen Time, uncategorized logs, >10 minute phone sessions.
- Reduce low-value apps and groups: delete, disable, turn off notifications, migrate to desktop, automate.
- Habits: + / - / = logs, routines, Atomic Habits four-factor improvements.
- OKR and nine-grid review: OKR Timeline, KR breakdown, Simplify / Passion / Organize, weekly highlights, Waiting follow-ups, Someday next tasks.
- TickTick statistics, summary, habit check-ins.
- Knowledge capability growth: hard knowledge, soft knowledge, progress.
- Health: weekly work hours around 50 as ideal, avoid above 70; focus on efficiency.
- Reading: work notes, book notes, reviews, Anki, reading-driven action plans, personality optimization, one weekly reading theme.
- Personal summary: negative feedback and improvements.
- Public output and reading ambitions: blog weekly if relevant, book review if relevant.

## Get Future

- Plan P1/P2 tasks with SMART and 2/8 principles.
- Review next week's calendar and adjust tasks.
- Arrange work through Thursday; use Friday for next-week preparation.
- Review Someday/Maybe and filter out low-importance items.
- Set goals at 120% challenge difficulty.
- Predict and prevent risks one week ahead.
- Review and adjust finances.
