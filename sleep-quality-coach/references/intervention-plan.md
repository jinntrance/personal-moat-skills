# Sleep Intervention Plan

This plan is based on the user's source image: a prioritized daily checklist for improving deep sleep from a low baseline, specifically mentioning 43 minutes of deep sleep.

## Default Daily Checklist

| Time | Action | Purpose |
| --- | --- | --- |
| 07:00-08:00 | Get outdoor sunlight for 20-30 minutes. | Anchor circadian rhythm and start nighttime melatonin countdown. |
| Before 14:00 | Stop caffeine from coffee, tea, cola, energy drinks, and similar sources. | Let adenosine pressure accumulate before bedtime. |
| 17:00-19:00 | Do 30-45 minutes of moderate to vigorous aerobic exercise. | Increase physical sleep drive and support slow-wave sleep. |
| 18:00-19:30 | Include moderate complex carbohydrates such as oats, sweet potato, rice, or whole grains at dinner. | Support evening insulin response and tryptophan transport. |
| Bedtime -90 min | Reduce blue light or wear blue-light blocking glasses; dim lights. Optional: magnesium glycinate or threonate, plus 3g glycine if appropriate. | Protect melatonin rhythm and reduce body temperature/arousal. |
| Bedtime -60 min | Take a 40-42 C warm shower or foot soak for about 15 minutes. | Create a post-bath cooling effect that can promote sleepiness. |
| Bedtime -30 min | Cool bedroom toward about 18 C if comfortable; play pink noise if useful. | Support deep sleep temperature and acoustic stability. |
| Bedtime | Do NSDR or 4-7-8 breathing. | Shift toward parasympathetic dominance and reduce sleep latency. |

## Personalization Rules

- If the user wakes later than 08:00, still prioritize outdoor sunlight within 60 minutes of waking.
- If exercise close to bedtime feels stimulating, move it earlier and keep only gentle stretching after dinner.
- If warm showers make the user alert, reduce temperature, shorten duration, or move the shower earlier.
- If bedroom cooling to 18 C is uncomfortable, aim for the coolest comfortable range and use breathable bedding.
- If pink noise is irritating or causes monitoring behavior, remove it.
- If the user has reflux, large late dinners, alcohol, or nicotine, address those before adding complex routines.
